# 🎮 Persona 3 Inspired Personal Site

## 📌 Overview
This project is a Persona 3-inspired personal portfolio website focused on interactive UI, animated frontend experiences, and responsive web design.

The website explores game-inspired interface aesthetics combined with modern frontend concepts and motion-based interactions.

---

## 🎯 Purpose
- Practice frontend development skills  
- Explore UI/UX inspired by game interfaces  
- Improve responsive web design and animation workflows  
- Experiment with AI-assisted frontend prototyping

---

## 🛠 Tools & Technologies
- HTML
- CSS
- JavaScript
- Vercel
- Figma
- AI-assisted workflow tools

- Researched Persona 3 interface aesthetics and visual direction
